Visualisation

------HOW TO VISUALISE--------

- Download the Visualisation Folder.
- Open the html file on website
- Upload medical image 
- View the original image, attained windowed image and default windowed image.
- Go through slices of 3D medical image by link or right button.
- Control the slider to check different slice of the image uploaded.
- Check the pixel area according to bar chart--highlight function
